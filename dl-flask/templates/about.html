Robotic Arm

In this project robotic-arm has been assembled with various part to create
a replica of human-arm. There are five servo motors attached to the joints of
robotic-arm, which are controlled by servo controller.



BCI

BCI-Brain Computer Interface, this part of project involves connection of
brain sensor with bluettoth module HC-05, which is further attched to servo
controler.



Neural Network

A deep learning model has been created for the project with the help of keras
and tensorflow. The model includes three inputs and 5 outputs.
